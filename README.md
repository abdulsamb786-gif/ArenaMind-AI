# ArenaMind AI

**Predict. Prevent. Protect.**

AI Mission Control for FIFA World Cup 2026 — an AI-native stadium operations platform where every recommendation, prediction, and decision is generated by Generative AI.

---

## Problem

Stadium operations today rely on manual coordination, static dashboards, and human intuition. During the FIFA World Cup 2026, 80,000+ fans per match create complexity that traditional systems cannot handle:

- Crowd congestion escalates without early warning
- Incident response depends on manual reporting
- Multilingual communication is fragmented
- Organizers lack real-time decision support
- Emergency response is reactive, not predictive

---

## Solution: ArenaMind AI

ArenaMind AI transforms stadium data into real-time operational decisions using Generative AI. It acts as an **AI Operations Commander** — not a chatbot.

### 5 Core Features

| # | Feature | What AI Does |
|---|---------|-------------|
| 1 | **AI Fan Copilot** | Remembers seat, language, preferences → guides, translates, recommends food, handles lost & found |
| 2 | **AI Mission Control** | Live dashboard with AI-generated insights + risk scoring + confidence explanations + live timeline |
| 3 | **AI Incident Workflow** | 6-step pipeline: Understand → Retrieve (RAG) → Predict → Recommend → Explain → Execute |
| 4 | **AI Scenario Simulator** | "What if Gate A closes?" → AI simulates crowd redistribution, risk, volunteer plan, announcement |
| 5 | **AI Executive Briefing** | One-click → AI generates operational summary with crowd, security, medical, transport, risks |

## AI Workflow Pipeline

Every incident or operational event follows this 6-step AI pipeline:

```
Input: "Child crying near Gate B"
    │
    ▼
┌──────────────────────────────────────────┐
│  1. AI Understands                       │
│     → Category: incident                 │
│     → Severity: high                     │
│     → Entities: child, Gate B            │
├──────────────────────────────────────────┤
│  2. AI Retrieves (RAG)                   │
│     → FIFA Lost and Found Protocol §3.2  │
│     → FIFA Child Safety Guidelines §2.1  │
├──────────────────────────────────────────┤
│  3. AI Predicts                          │
│     → Risk Score: 92                     │
│     → Time to critical: 12 min           │
│     → Affected fans: 50                  │
├──────────────────────────────────────────┤
│  4. AI Recommends                        │
│     → Activate Lost Child Protocol       │
│     → Tasks: Secure, Broadcast, Guide    │
│     → Volunteers needed: 3               │
├──────────────────────────────────────────┤
│  5. AI Explains                          │
│     → Primary reason + factors           │
│     → What happens if no action taken    │
│     → Confidence breakdown               │
├──────────────────────────────────────────┤
│  6. AI Executes                          │
│     → Mission created                    │
│     → Announcement generated             │
│     → Dashboard updated                  │
│     → Timeline updated                   │
└──────────────────────────────────────────┘
```

## Features in Detail

### 1. AI Fan Copilot
- Natural conversation with seat memory
- Personalized food recommendations based on seat location
- Multilingual support: English, Hindi, Spanish, French, Arabic, Portuguese
- Wheelchair-friendly route guidance
- Lost & Found assistance
- Emergency help

### 2. AI Mission Control
- Live stadium dashboard with real-time WebSocket updates
- AI-generated insights on every card
- Crowd density visualization with gate-level occupancy
- Transport, parking, medical, weather status
- **AI Confidence Layer** — every recommendation shows confidence % + reasons
- **AI Timeline** — live event log of decisions
- **AI Agent Status** — 6 agents with live online/offline indicators

### 3. AI Incident Workflow
- Full 6-step AI pipeline visualization
- RAG policy retrieval with FIFA document references
- Risk scoring with time-to-critical prediction
- Structured response plan with tasks, volunteers, ETA
- AI explanation with supporting factors
- Multilingual public announcement generation
- Confidence breakdown (data quality, pattern match, historical accuracy)

### 4. AI Scenario Simulator
- **Judge Demo Mode** — 5 preloaded scenarios (Lost Child, Medical, Rain, Congestion, Transport)
- **What-If Simulation** — Close Gate A, Heavy Rain, Crowd Surge
- AI predicts crowd redistribution, risk scores, volunteer requirements
- Generates complete operational response plans

### 5. AI Executive Briefing
- One-click operational summary
- Crowd, security, medical, volunteer, transport summaries
- Key risks identified by AI
- AI recommendations for next actions
- Next hour outlook
- Generated on demand for command leadership

## Setup

### Prerequisites
- Node.js 18+
- Google Gemini API key (optional — runs in offline demo mode without one)

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd ArenaMind-AI

# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install
```

### Configuration

```bash
# In server/.env
PORT=3001
GEMINI_API_KEY=your-gemini-api-key-here
CLIENT_URL=http://localhost:5173
```

### Running

```bash
# Terminal 1: Backend
cd server
npm start

# Terminal 2: Frontend
cd client
npm run dev
```

The app will be available at `http://localhost:5173`.
